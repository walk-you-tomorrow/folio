### 구성 요소
| 요소 | 스타일 | 비고 |
|---|---|---|
| 로고 | Folio 텍스트 로고, `#1A1A2E` | 좌측 상단 |
| 태그라인 | 28-40px, Bold, `#1A1A2E` | 중앙 정렬 |
| 서브 카피 | 15-17px, Regular, `#6B6B80` | 태그라인 아래 |
| CTA 버튼 | `#3B6FFF`, 텍스트 white, radius 8px, 높이 48px | 한 화면에 1개만 |
| 배경 | `#F7F7F5` Canvas White | 전체 |

### 인터랙션
- CTA 호버: `#3B6FFF` → `#2E5CE6` (약간 어둡게), `transform: scale(1.02)`
- CTA 클릭: POST `/api/canvas` → 생성 후 `/{canvasId}`로 리다이렉트
- 로딩 중: 버튼 텍스트 → "캔버스 펼치는 중이에요..." + 스피너

### 반응형
| 브레이크포인트 | 변경 사항 |
|---|---|
| Desktop (1024px+) | 기본 레이아웃, 좌우 여백 넓게 |
| Tablet (768-1023px) | 여백 축소, 일러스트 크기 조절 |
| Mobile (< 768px) | 단일 컬럼, 태그라인 24px, CTA 풀 width |

## S2. 캔버스 (메인 작업 화면)

### 와이어프레임
```
┌─────────────────────────────────────────────────┐
│ [Folio]   "제목 없는 캔버스"      [👤👤 3명] [공유] │ ← 상단 바
├─────────────────────────────────────────────────┤
│                                                  │
│                                                  │
│          ┌──────────┐                            │
│          │ 텍스트    │                            │
│          │ 스티커    │     ┌────────────┐         │
│          └──────────┘     │            │         │
│                           │   이미지    │         │
│     ┌──────────┐          │            │         │
│     │ 또 다른   │          └────────────┘         │
│     │ 스티커    │                                 │
│     └──────────┘                                 │
│                                                  │
│                                                  │
│                                                  │
│                                                  │
│        ┌──────────────────────────┐              │
│        │  [T] 텍스트  [📷] 이미지  │ [✋] 패닝     │ ← 하단 툴바
│        └──────────────────────────┘              │
│                                          [100%]  │ ← 줌 컨트롤
└─────────────────────────────────────────────────┘
```

### 상단 바 (Top Bar)
| 요소 | 스타일 | 인터랙션 |
|---|---|---|
| 로고 | "Folio" 텍스트, 클릭 → 홈 | 좌측 |
| 캔버스 제목 | 15px, `#6B6B80`, 클릭 시 편집 가능 | 중앙 |
| 접속자 표시 | 원형 아바타(컬러), 최대 5개 + "+N" | 우측 |
| 공유 버튼 | `#3B6FFF`, "공유" 텍스트, radius 8px | 최우측 |

- 상단 바 높이: 56px
- 배경: `#FFFFFF`, 하단 border `1px solid #EEEEF0`

### 캔버스 영역
| 항목 | 스펙 |
|---|---|
| 배경 | `#F7F7F5` Canvas White |
| 그리드 | 미표시 (MVP) |
| 커서 | 기본 → 패닝 모드: grab/grabbing |

### 텍스트 스티커 컴포넌트
```
┌────────────────────────┐
│  스티커 텍스트 내용      │  ← 배경색: 6가지 프리셋
│  여러 줄 가능           │
│                        │  ← 모서리 radius: 6px
│               [색상][✕] │  ← 선택 시만 표시
└────────────────────────┘
    ↕ 리사이즈 핸들
```

**스티커 배경색 프리셋:**
| 이름 | Hex | 용도 |
|---|---|---|
| Yellow | `#FFF9DB` | 기본 (아이디어) |
| Blue | `#DBE4FF` | 정보, 참고 |
| Green | `#D3F9D8` | 완료, 합의 |
| Pink | `#FFE0E6` | 중요, 주의 |
| Purple | `#E8DAFB` | 질문 |
| Gray | `#F1F3F5` | 기타 |

**스티커 상태:**
| 상태 | UI |
|---|---|
| 기본 | 그림자 없음, 배경색만 |
| 호버 | `box-shadow: 0 2px 8px rgba(0,0,0,0.08)` |
| 선택 | 파란 테두리 `2px solid #3B6FFF`, 핸들 표시 |
| 편집 중 | 커서 깜빡임, 테두리 유지 |
| 다른 사용자 편집 중 | 해당 사용자 컬러 테두리, "OO님이 수정 중" |

### 이미지 컴포넌트
```
┌──────────────────────┐
│                      │
│      [이미지]         │  ← 원본 비율 유지
│                      │
│                  [✕]  │  ← 선택 시만 표시
└──────────────────────┘
 ↔ ↕ 모서리 리사이즈 핸들
```

**이미지 상태:**
| 상태 | UI |
|---|---|
| 업로드 중 | placeholder + 프로그레스 바 |
| 로드 완료 | 이미지 표시, radius 4px |
| 선택 | 파란 테두리, 4모서리 리사이즈 핸들 |
| 에러 | "이미지를 불러오지 못했어요" placeholder |

### 하단 툴바 (Bottom Toolbar)
```
┌────────────────────────────────────────────┐
│   [T 텍스트]   [🖼 이미지]   │   [✋ 이동]    │
└────────────────────────────────────────────┘
```

| 항목 | 스펙 |
|---|---|
| 위치 | 하단 중앙, 캔버스 위에 float |
| 배경 | `#FFFFFF`, radius 12px, shadow `0 2px 8px rgba(0,0,0,0.08)` |
| 높이 | 48px |
| 아이콘 | 24px, `#6B6B80` 기본, `#3B6FFF` 활성 |
| 활성 상태 | 배경 `#EBF0FF`, 아이콘 `#3B6FFF` |

```

│  [-]     │
│  100%   │
```

- 위치: 우측 하단
- 배경: `#FFFFFF`, radius 8px
- 줌 퍼센트: 13px, `#6B6B80`

## 마이크로 인터랙션

| 인터랙션 | 애니메이션 |
|---|---|
| 스티커 생성 | `scale(0.8) → scale(1)`, 200ms ease-out |
| 스티커 삭제 | `opacity 1→0 + scale(1→0.8)`, 150ms |
| 모달 열기 | `opacity 0→1 + translateY(8px→0)`, 200ms |
| 모달 닫기 | `opacity 1→0`, 150ms |
| 토스트 표시 | 하단에서 올라옴, `translateY(100%→0)`, 300ms |
| 줌 변경 | `transition: transform 200ms ease-out` |
| 이미지 업로드 | 프로그레스 바 + 완료 시 체크 아이콘 300ms |
| 접속자 입장 | 아바타 `scale(0→1)` + 바운스 |
| 접속자 퇴장 | 아바타 `opacity 1→0`, 300ms |
| 버튼 호버 | `background-color` 200ms transition |
| 드래그 | 살짝 `elevation` 증가, shadow 강화 |

## 접근성 고려사항

| 항목 | 구현 |
|---|---|
| 키보드 포커스 | 모든 인터랙티브 요소에 visible focus ring (`#3B6FFF` 2px) |
| aria-label | 버튼, 아이콘, 입력 필드에 적용 |
| 역할 정의 | 툴바: `role="toolbar"`, 모달: `role="dialog"` |
| 포커스 트랩 | 모달 오픈 시 모달 내부로 포커스 제한 |
| 색상 대비 | 텍스트 `#1A1A2E` on `#F7F7F5` = 13.5:1 (AAA) |
| 스크린 리더 | 상태 변경 시 `aria-live="polite"` 알림 |
| 모션 감소 | `prefers-reduced-motion` 미디어 쿼리 대응 |
